# sprint-four
This project contains a basic analysis on a vehicle advertisment dataset. Using libraries such as Streamlit to build the web app, Plotly Express to create charts, and Pandas to load and handle data. We explore and visualize the vehicles data using random sampling and filters.

To run this project follow these steps:

1.Open terminal and find project directory.

2. (Optional but recommended) Activate the projects virtual enviorment: 

*For macOs/Linux users: sourse venv/bin/activate

*For Windows users: venv\Scripts\activate

3. Install required packages if they're not already installed: pip install -r requirments.txt 

4. Start the Streamlit application: python-m sttreamlit run app.py

5. Open the local URL  in terminal (typically http://localhost:8501) in your web browser.
